# 📝 Form Validation

## 📌 Project Overview

This project is a simple and responsive **Registration Form Validation** built using **HTML, CSS, and JavaScript**. It validates user input before submission and provides real-time error messages for invalid fields.

## 🚀 Features

* User-friendly registration form
* Required field validation
* Email format validation
* Password minimum length validation
* Confirm password matching
* Red error messages with highlighted input fields
* Error messages disappear while typing
* Success popup after valid submission
* Form automatically resets after successful submission

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```
form-validation/

│── README.md
```

## 📖 Validation Rules

* Name field cannot be empty.
* Email must contain a valid email format.
* Password must contain at least 8 characters.
* Confirm Password must match the Password field.
* All fields are required before submission.

## ▶️ How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser.
4. Fill in the form and test the validation.
